# esri - geometry - java
## 简介
本项目主要讲解的内容为 esri-geometry-api jar的使用以及geometry的一些基础知识

## 依赖
- Java JDK 1.6或更高版本。
- maven 3.6或更高版本。
```xml
<dependency>
  <groupId>com.esri.geometry</groupId>
  <artifactId>esri-geometry-api</artifactId>
  <version>2.2.2</version>
</dependency>
```
